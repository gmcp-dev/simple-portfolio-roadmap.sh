# Simple Portfolio Website

Personal portfolio of **Gerald Caraballo** — Full-Stack Developer.  
Roadmap project: [Basic HTML Website](https://roadmap.sh/projects/basic-html-website).

## Pages

| Route            | Description                    |
|------------------|--------------------------------|
| `/`              | Home — work experience & education |
| `/projects.html` | Projects                       |
| `/articles.html` | Articles                       |
| `/contact.html`  | Contact form                   |

## Tech Stack

- Semantic HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript
- No frameworks or external dependencies

## Project Structure

```
├── index.html
├── projects.html
├── articles.html
├── contact.html
├── layouts/
│   ├── nav.html
│   └── footer.html
└── src/
    ├── css/styles.css
    └── js/scripts.js
```

## Usage

Open any `.html` file in your browser.  
The nav and footer are loaded dynamically via `fetch()`.
